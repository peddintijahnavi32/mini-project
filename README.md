# Django To-Do List App

A simple To-Do List application built using Django and MySQL.

## Features

* Add tasks
* Update tasks
* Delete tasks
* Mark tasks as completed
* Django Admin Panel

## Technologies Used

* Python
* Django
* MySQL
* HTML
* CSS

## Installation

1. Clone the repository
2. Install dependencies

pip install -r requirements.txt

3. Run migrations

python manage.py migrate

4. Start the server

python manage.py runserver
